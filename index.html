Book Recommender System (BSAN 780)

For our capstone in BSAN 780, my team and I designed a hybrid book recommender system using SQLite and Python. We implemented both content-based and collaborative filtering using cosine similarity, built a user-item ratings matrix, and generated personalized book recommendations. I led much of the data preprocessing, cleaning, and recommendation logic development across SQL and Python. Our final system was fully functional, interpretable, and capable of handling cold-start users.
